# Pacman Game (Java OOP)

A simple console-based Pacman Game developed using Java and Object-Oriented Programming (OOP) concepts. In this game, the player controls Pacman, moves around the game board, collects food, and avoids being caught by the ghost.

## Features
- Pacman movement using user input (W, A, S, D)
- Ghost automatically chases Pacman
- Food collection system
- Win and Game Over conditions
- Console-based game board display

## OOP Concepts Used
- Classes and Objects
- Encapsulation
- Constructors
- Method Interaction

## Classes Included
- `PacmanGame` – Main class and game loop  
- `GameBoard` – Displays the board  
- `Pacman` – Player movement  
- `Ghost` – Enemy movement  
- `Food` – Winning condition  

## How to Run
1. Compile the program:
```bash
javac PacmanGame.java
```

2. Run the program:
```bash
java PacmanGame
```

## Controls
- W = Move Up  
- S = Move Down  
- A = Move Left  
- D = Move Right  

## Objective
Reach the food before the ghost catches Pacman.
